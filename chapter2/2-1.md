# 切片

> 切片相对于 字符序列（字符串、list、tuple、dict、set）来说很方便



```python
L = ['zhangsan','lisi','wangwu','zhaoliu','lucy','tom']

#取前三个元素 [)
L[0:3]

#如果第一个索引是0 可以省略
L[:3]

#倒数切片
L[-2:]
L[-2:-1]
```



倒数第一个元素的索引是 `-1`

```python
#切片操作十分有用
L = list(range(100))

#前10个数
L[:10]

#后10个数
L[-10:]

#11-20
L[10:20]

#前10个 每2个取1个
L[:10:2]

#所有数 每5个取1个
L[::5]

#什么不写 复制一个list
L[:]
```



字符串也可以切片

```python
S = 'ABCDEF'
S[:3]
S[::2]
```
